Superstore Rush

Play Git

Super store store rush is an arcade idle hypercasual game created with the unity game engine. This game explores the latest arcade idle genre that's recently hit the market. It focuses on owning a store and expanding parts of the store by unlocking shelves, rooms and other components that make up a supermarket. This mechanic has been used by several studios and has proven to boost player retention. Superstore Rush has several core features like AI navigation, stacking and unstacking items, unlocking, IAP, player feedback and so much more. The core loop is picking up packages from a truck and stocking your shelves with products, customers gather this items and require the player at the register to sell items and collect cash in return. Earn money to expand store and the loop continues.
